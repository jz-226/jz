## 📝 一句话简介

一个先问你是谁、再帮你规划行程的 AI Agent——3 种画像 × 3 种模式，拒绝千篇一律的攻略。

---

## 📖 应用概述

「旅行规划助手」是一款基于大模型 + RAG 知识库的智能旅行规划 Agent。它不做"景点列表生成器"——它会在规划前先跟你聊几句，搞清楚你是谁、预算多少、喜欢什么节奏，再结合本地知识库，给你一份**能真正落地的专属行程**。

同样是"去苏州玩两天"：
- 学生会得到一张免费景点 + 青旅床位 + 学生证折扣的精打细算版
- 白领会得到苏绣体验 + 精品民宿 + 地道苏帮菜的品质版
- 带父母出行的会得到轮椅友好路线 + 午休安排 + 评弹茶馆的舒适版

没有万能行程，只有懂你的行程。

---

## ✨ 核心功能

- **🎯 对话式画像识别**：不是填表，是聊天。像朋友一样自然问你几个人、几天、预算多少，然后自动匹配规划模式
- **📚 本地知识增强**：搭载 RAG 知识库，路线、门票、交通、美食都基于真实本地信息，不是模型凭空编的
- **💰 透明预算控制**：每份行程附带分项预算明细——门票多少、交通多少、三餐多少——每一笔都算清楚
- **🛡️ 诚实规划**：预算不切实际时不编造方案，而是算账告诉你为什么不行，再给替代思路。拒绝比糊弄更有价值
- **♿ 无障碍关怀**：老人、行动不便者自动匹配平坦景点和休息节奏
- **📋 完整出行包**：行程 + 预算 + 交通 + 住宿 + 天气提醒 + 避坑指南 + 应急信息，七个板块一份搞定

## 🛠️ 技术架构

- **对话引擎**：DeepSeek V4
- **应用框架**：Dify（Bot + RAG 知识库 + Prompt 编排）
- **Prompt 架构**：4 层架构（角色 / 规则 / 约束 / 示例）+ SOP 对话流程 + Few-shot
- **RAG**：本地文档 Embedding 向量检索，可按需接入任意目的地知识库
