---

## 📖 应用概述

「苏州旅行规划助手」是一款基于大模型 + RAG 知识库的智能旅行规划 Agent，专为前往苏州的游客提供**千人千面**的行程规划服务。

和通用 AI 聊天机器人不同，这个助手不只是"列景点清单"——它会像朋友一样先了解你是谁、怎么玩、预算多少，再结合真实的苏州本地知识库（景点信息、门票价格、交通路线、地道美食），为你量身定制一份**能真正落地的专属行程**。

无论是学生党极限穷游、白领周末品质游，还是带父母长辈慢游苏州，这个助手都能给出贴合你节奏的方案。

---

## ✨ 核心功能

- **🎯 智能画像识别**：对话式收集用户信息，自动匹配 3 种画像（学生 / 白领 / 老人）× 3 种模式（穷游打卡 / 周末深度 / 慢游辅助），而不是甩给你通用模板
- **🏯 本地知识增强**：搭载苏州本地 RAG 知识库，覆盖园林门票、公交路线、地道小吃、预约规则，信息准确可落地
- **💰 精确预算控制**：行程附带分项预算明细表，从门票到交通到三餐，每一笔钱都有出处，帮你在预算内玩到最好
- **🧓 无障碍关怀**：针对老人等特殊群体，自动推荐平坦好走的景点、标注休息点和无障碍设施
- **🛡️ 诚实规划**：预算不切实际时不会编造虚假方案，而是诚实告知并给出替代建议——拒绝有时是更好的用户体验
- **📋 完整出行包**：输出包含 4 大板块（行程 / 预算 / 交通 / 住宿）+ 天气提醒 + 避坑指南 + 应急信息

## 🛠️ 技术架构

- **对话引擎**：DeepSeek V4
- **应用框架**：Dify（Bot + RAG 知识库 + Prompt 编排）
- **知识库**：5 份苏州本地旅行资料，Embedding 向量检索
- **Prompt 架构**：4 层结构（角色层 / 规则层 / 约束层 / 示例层）+ SOP 对话流程 + Few-shot 示例
