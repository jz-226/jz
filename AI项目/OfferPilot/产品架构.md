# OfferPilot · 产品架构

## 产品定位

AI Offer 操作系统。不是 Chatbot，不是学习计划生成器。围绕用户目标运行的长期陪伴型 AI 产品。

## 核心差异化

| ChatGPT | OfferPilot |
|------|------|
| 回答问题 | 追踪目标 |
| 生成学习计划后结束 | 每天陪你执行计划 |
| 不知道你昨天做了什么 | 记住你的全部成长历史 |
| 你主动问它 | 它主动问你 |
| 对话结束就清零 | 飞书表格持久化存储 |

## 技术架构

```
用户端（Dify Chatbot / 未来 App）
    │
    ▼
┌─────────────────────────────┐
│  OfferPilot Core             │
│                              │
│  · 目标管理                   │
│  · 每日追踪                   │
│  · 能力评估                   │
│  · 周报生成                   │
│  · 面试记录                   │
│                              │
│  Model Router                 │
│  ├─ DeepSeek（日常对话）       │

           │
           ▼

```

## 数据模型

### goals 表

| 字段 | 类型 | 说明 |
|------|------|------|
| 目标 | 文本 | "2027苏州AI PM" |
| 阶段 | 单选 | 学习/作品集/投递/面试 |
| 截止日期 | 日期 | 目标时间 |
| 每日时间 | 数字 | 小时 |

### daily_logs 表

| 字段 | 类型 | 说明 |
|------|------|------|
| 日期 | 日期 | 自动 |
| 做了什么 | 多行文本 | 用户回答 |
| 进展 | 单选 | 近了/原地踏步/退了 |
| 备注 | 文本 | Agent 分析 |

### interviews 表

| 字段 | 类型 | 说明 |
|------|------|------|
| 公司 | 文本 | |
| 岗位 | 文本 | |
| 日期 | 日期 | |
| 表现自评 | 单选 | 好/一般/不好 |
| 关键问题 | 多行文本 | 面试官问了什么 |

## 未来 Agent 矩阵

| Agent | 职责 |
|------|------|
| Goal Agent | 管理目标+每日追踪 |
| Analysis Agent | 分析JD+能力差距 |
